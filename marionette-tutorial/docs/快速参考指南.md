# Marionette 快速参考指南

## 核心类和方法速查表

### datamodule.py

| 类/方法 | 功能 | 输入 | 输出 |
|---------|------|------|------|
| `Sequence.__init__` | 创建轨迹对象 | time, conditions, tmax | Sequence |
| `Batch.from_sequence_list` | 创建批次 | List[Sequence] | Batch |
| `DataModule.setup` | 准备数据 | - | - |
| `DataModule.train_dataloader` | 训练数据加载器 | - | DataLoader |

### add_thin/diffusion/model.py

| 类/方法 | 功能 | 输入 | 输出 |
|---------|------|------|------|
| `AddThin.noise` | 前向扩散 | x0, n | xn, x0_thin |
| `AddThin.forward` | 训练前向 | x0 | logits, log_like, xn |
| `AddThin.sample` | 生成序列 | n_samples, x_n, tmax | x0 |
| `AddThin.sample_posterior` | 反向采样一步 | xn, n | xn-1 |

### discrete_diffusion/diffusion_transformer.py

| 类/方法 | 功能 | 输入 | 输出 |
|---------|------|------|------|
| `DiffusionTransformer.q_sample` | 前向扩散 | log_x_start, t, batch | log_xt |
| `DiffusionTransformer.predict_start` | 预测原始 | log_x_t, cond_emb, t | log_pred |
| `DiffusionTransformer.p_sample` | 反向采样 | log_x, cond_emb, t | log_sample |
| `DiffusionTransformer.sample_fast` | 快速生成 | batch | Batch |

### tasks.py

| 类/方法 | 功能 | 输入 | 输出 |
|---------|------|------|------|
| `DensityEstimation.training_step` | 训练步骤 | batch | loss |
| `DensityEstimation.configure_optimizers` | 配置优化器 | - | optimizers |

## 常用命令

### 训练
```bash
# 基础训练
python train.py

# 指定配置
python train.py --config-name train

# 覆盖参数
python train.py data.batch_size=256 trainer.max_epochs=50
```

### 采样
```bash
# 使用 W&B run ID
python sample.py --run_id abc123

# 指定输出路径
python sample.py --run_id abc123 --output ./outputs/
```

### 评估
```bash
# 单个任务
python evaluation.py --task LocRec --datasets Istanbul

# 所有任务（并行）
bash sample_evaluation.sh abc123
```

## 常见错误与解决

### CUDA Out of Memory
```python
# 解决方法：
# 1. 减小 batch_size
# 2. 使用梯度累积
# 3. 减少 diffusion steps
```

### Key Error: 'sequences'
```python
# 解决方法：检查 .pkl 文件格式
import pickle
with open("Istanbul.pkl", "rb") as f:
    data = pickle.load(f)
print(data.keys())  # 应该包含 'sequences'
```

### NaN Loss
```python
# 解决方法：
# 1. 检查学习率（可能过大）
# 2. 检查数据（是否有异常值）
# 3. 使用梯度裁剪
```

## 性能优化

### 加速训练
```yaml
# config/train.yaml
trainer:
  accumulate_grad_batches: 4  # 梯度累积
  precision: 16               # 混合精度
data:
  num_workers: 4              # 数据加载并行
  pin_memory: true            # 加速 CPU→GPU 传输
```

### 加速采样
```python
# 减少扩散步数
tpp_model.steps = 50  # 默认 100
discrete_diffusion.diffusion_steps = 100  # 默认 200
```
