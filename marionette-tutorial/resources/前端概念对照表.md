## 🎯 核心概念详解

### 1. 数据结构

#### 数组的演进

```javascript
// JavaScript/前端
const arr = [1, 2, 3];
const objArr = [{id: 1, name: "A"}];
```

```python
# Python 基础
arr = [1, 2, 3]  # List，可以存任意类型
obj_dict = {"id": 1, "name": "A"}  # Dict

# NumPy (数值计算)
import numpy as np
arr_np = np.array([1, 2, 3])  # 高性能数值数组

# PyTorch (深度学习)
import torch
arr_torch = torch.tensor([1, 2, 3])  # GPU 可计算的张量
```

#### 关键区别

| 特性 | JS Array | Python List | np.ndarray | torch.Tensor |
|------|----------|-------------|------------|--------------|
| 类型 | 任意 | 任意 | 固定类型 | 固定类型 |
| 性能 | 低 | 低 | 高 | 最高 (GPU) |
| GPU | ❌ | ❌ | ❌ | ✅ |

### 2. 函数与类

#### 函数定义

```javascript
// JavaScript
function add(a, b) {
    return a + b;
}

// 箭头函数
const add = (a, b) => a + b;
```

```python
# Python
def add(a, b):
    return a + b

# Lambda 函数 (类似箭头函数)
add = lambda a, b: a + b
```

#### 类的定义

```javascript
// JavaScript (ES6+)
class Person {
    constructor(name) {
        this.name = name;
    }

    greet() {
        return `Hello, ${this.name}`;
    }
}
```

```python
# Python
class Person:
    def __init__(self, name):  # 类似 constructor
        self.name = name

    def greet(self):  # self 是必需的
        return f"Hello, {self.name}"
```

### 3. 异步编程

```javascript
// JavaScript (Promise)
fetch('/api/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

// async/await
async function getData() {
    try {
        const response = await fetch('/api/data');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}
```

```python
# Python (Generator 类似 Promise)
def fetch_data():
    # 模拟异步操作
    yield {"data": "value"}

for data in fetch_data():
    print(data)

# asyncio (真正的异步)
import asyncio

async def get_data():
    await asyncio.sleep(1)  # 类似 await fetch()
    return {"data": "value"}

# 运行
data = asyncio.run(get_data())
```

## 🧠 AI 特有概念

### 张量 (Tensor)

> 前端类比：**多维数组**，但可以在 GPU 上并行计算

```python
# 0D 张量 (标量)
scalar = torch.tensor(3.14)  # 类似 const x = 3.14;

# 1D 张量 (向量)
vector = torch.tensor([1, 2, 3])  # 类似 const arr = [1, 2, 3];

# 2D 张量 (矩阵)
matrix = torch.tensor([[1, 2], [3, 4]])  # 类似 const mat = [[1,2], [3,4]];

# 3D 张量 (RGB 图像)
image = torch.tensor([[[255, 0, 0], ...]])  # 通道×高×宽

# 4D 张量 (图像批次)
batch = torch.tensor([...])  # 批次×通道×高×宽
```

### 批次 (Batch)

> 前端类比：**批量处理数据**，类似 Promise.all 但并行计算

```javascript
// 前端：串行处理
const results = [];
for (const item of items) {
    results.push(await processItem(item));
}

// 前端：并行处理
const results = await Promise.all(items.map(item => processItem(item)));
```

```python
# AI：批次处理 (GPU 并行计算)
batch = torch.stack([item1, item2, item3])  # shape: [3, ...]
outputs = model(batch)  # 一次处理 3 个样本
```

### 梯度 (Gradient)

> 前端类比：**自动微分**，类似计算复合函数的导数链

```javascript
// 前端：手动计算
function f(x) {
    return x * x;
}
// 导数是 2x，需要手动计算
```

```python
# AI：自动计算梯度
x = torch.tensor(2.0, requires_grad=True)
y = x * x
y.backward()  # 自动计算 dy/dx = 2x = 4
print(x.grad)  # tensor(4.)
```

### 训练循环 (Training Loop)

> 前端类比：**迭代优化**，类似不断调整参数逼近目标

```javascript
// 前端：手动优化参数
let param = 0;
for (let i = 0; i < 100; i++) {
    const error = target - param * input;
    param += learning_rate * error;
}
```

```python
# AI：使用优化器自动更新
optimizer = torch.optim.Adam([param], lr=0.001)
for epoch in range(100):
    loss = criterion(model(input), target)
    optimizer.zero_grad()
    loss.backward()  # 计算梯度
    optimizer.step()  # 更新参数
```

## 📊 常见操作对照

### 数据遍历

```javascript
// JavaScript
items.forEach(item => console.log(item));
for (const item of items) { }
items.map(item => item * 2);
items.filter(item => item > 0);
```

```python
# Python
for item in items:
    print(item)

# 列表推导式 (类似 map/filter)
[item * 2 for item in items]
[item for item in items if item > 0]

# NumPy/Pandas 向量化操作 (更快)
arr * 2  # 逐元素乘法
arr[arr > 0]  # 布尔索引
```

### 数据拼接

```javascript
// JavaScript
const combined = [...arr1, ...arr2];  // 展开运算符
```

```python
# Python
combined = arr1 + arr2  # List 拼接
combined = np.concatenate([arr1, arr2])  # NumPy
combined = torch.cat([arr1, arr2])  # PyTorch
```

### 对象/字典操作

```javascript
// JavaScript
const obj = { a: 1, b: 2 };
Object.keys(obj);  // ['a', 'b']
Object.values(obj);  // [1, 2]
obj.a = 3;  // 修改
delete obj.b;  // 删除
```

```python
# Python
obj = {'a': 1, 'b': 2}
obj.keys()  # dict_keys(['a', 'b'])
obj.values()  # dict_values([1, 2])
obj['a'] = 3  # 修改
del obj['b']  # 删除
```

## ⚠️ 常见陷阱

### 1. 索引从 0 开始 (相同)

```javascript
// JavaScript
arr[0]  // 第一个元素
```

```python
# Python (相同)
arr[0]  # 第一个元素
```

### 2. 切片语法 (不同！)

```javascript
// JavaScript
arr.slice(0, 2)  // [0, 1] - 不包含结束索引
```

```python
# Python (相同)
arr[0:2]  # [0, 1] - 不包含结束索引

# 但 Python 有更多特性
arr[::-1]  # 反转数组
arr[::2]  # 每隔一个取元素
```

### 3. 循环变量作用域

```javascript
// JavaScript (var 会有作用域问题)
for (var i = 0; i < 10; i++) { }
console.log(i);  // 10 (泄漏)

// 使用 let 解决
for (let i = 0; i < 10; i++) { }
console.log(i);  // ReferenceError
```

```python
# Python (没有这个问题)
for i in range(10):
    pass
print(i)  # 9 (循环后变量保留)
```

### 4. 可变/不可变类型

```javascript
// JavaScript (都是引用)
const arr = [1, 2, 3];
arr.push(4);  // 可以修改！

// 使用 Object.freeze 冻结
const frozen = Object.freeze([1, 2, 3]);
frozen.push(4);  // 报错
```

```python
# Python (分可变和不可变)
arr = [1, 2, 3]  # List，可变
arr.append(4)  # 可以修改

tup = (1, 2, 3)  # Tuple，不可变
tup.append(4)  # 报错！
```
