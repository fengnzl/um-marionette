## 🔍 常见错误速查

### 环境相关错误

#### 错误 1: ModuleNotFoundError

```python
ModuleNotFoundError: No module named 'torch'
```

**原因**: 依赖包未安装

**解决方案**:
```bash
# 检查是否安装
pip list | grep torch

# 重新安装
pip install torch==2.0.0
```

---

#### 错误 2: CUDA Out of Memory

```python
RuntimeError: CUDA out of memory. Tried to allocate 512.00 MiB
```

**原因**: GPU 显存不足

**解决方案**:
```python
# 方法 1: 减小 batch_size
# config/train.yaml
data:
  batch_size: 128  # 从 512 减小到 128

# 方法 2: 使用梯度累积
# config/train.yaml
trainer:
  accumulate_grad_batches: 4

# 方法 3: 清理 GPU 缓存
import torch
torch.cuda.empty_cache()
```

---

#### 错误 3: ImportError (版本不兼容)

```python
ImportError: cannot import name 'xyz' from 'abc'
```

**原因**: 包版本不匹配

**解决方案**:
```bash
# 检查已安装版本
pip show torch pytorch-lightning

# 按要求版本重新安装
pip install torch==2.0.0 pytorch-lightning==1.9.5 --force-reinstall
```

#### 错误 5: KeyError (数据格式问题)

```python
KeyError: 'sequences'
```

**原因**: .pkl 文件格式不符合预期

**解决方案**:
```python
import torch

# 检查 .pkl 文件内容
data = torch.load('data/Istanbul.pkl', map_location='cpu')
print("Keys in pkl file:")
for key in data.keys():
    print(f"  - {key}: {type(data[key])}")

# 预期应该包含:
# - sequences: List[Dict]
# - t_max: float
# - num_marks: int
# - num_pois: int
# - poi_gps: Dict
```

```python
RuntimeError: shape '[128, 50]' is invalid for input of size 6400
```

**解决方案**:
```python
# 打印张量形状

```

### 训练相关错误

#### 错误 7: NaN Loss

```python
loss: tensor(nan)
```

**原因**: 梯度爆炸或数值不稳定

**解决方案**:
```python
# 方法 1: 降低学习率
# config/train.yaml
task:
  learning_rate1: 0.0001  # 从 0.001 降低

# 方法 2: 使用梯度裁剪
# tasks.py
torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)

# 方法 3: 检查数据是否有异常值
print("Max value:", data.max())
print("Min value:", data.min())
print("Has NaN:", torch.isnan(data).any())
```

---

#### 错误 8: 代码语法错误

```python
SyntaxError: invalid syntax
```

**原因**: Python 语法错误（如使用了 JavaScript 语法）

**常见 JS → Python 语法错误**:
```javascript
// ❌ JavaScript 语法
const arr = [1, 2, 3];
arr.map(x => x * 2);
```

```python
# ✅ Python 语法
arr = [1, 2, 3]
result = [x * 2 for x in arr]
```

## 📊 性能分析

### 测量执行时间

```python
import time

# 方法 1: 基础计时
start = time.time()
result = slow_function()
elapsed = time.time() - start
print(f" Took {elapsed:.2f} seconds")

# 方法 2: 使用 timeit
import timeit
time_taken = timeit.timeit(lambda: slow_function(), number=10)
print(f"Average time: {time_taken/10:.4f} seconds")

# 方法 3: PyTorch GPU 计时
start = torch.cuda.Event(enable_timing=True)
end = torch.cuda.Event(enable_timing=True)

start.record()
result = model(batch)
end.record()

torch.cuda.synchronize()
print(f"GPU time: {start.elapsed_time(end)} ms")
```

## 🐛 常见陷阱与解决方案

### 陷阱 1: 可变默认参数

```python
# ❌ 错误
def foo(items=[]):
    items.append(1)
    return items

# ✅ 正确
def foo(items=None):
    if items is None:
        items = []
    items.append(1)
    return items
```

---

### 陷阱 2: 浅拷贝 vs 深拷贝

```python
# ❌ 浅拷贝（只复制引用）
list2 = list1.copy()

# ✅ 深拷贝（复制所有内容）
import copy
list2 = copy.deepcopy(list1)
```

---

### 陷阱 3: 整数除法

```python
# Python 3 中除法自动是浮点数
result = 5 / 2  # 2.5

# 使用 // 进行整数除法
result = 5 // 2  # 2
```

---

### 陷阱 4: 字符串不可变

```python
# ❌ 字符串不能修改
s = "hello"
s[0] = "H"  # TypeError

# ✅ 需要创建新字符串
s = "H" + s[1:]  # "Hello"
```

### 输入验证

```python
def train_model(config: dict) -> None:
    # 验证必需参数
    required_keys = ['batch_size', 'epochs', 'learning_rate']
    for key in required_keys:
        if key not in config:
            raise ValueError(f"Missing required config key: {key}")

    # 验证参数范围
    if config['epochs'] <= 0:
        raise ValueError("Epochs must be positive")

    # 验证类型
    if not isinstance(config['batch_size'], int):
        raise TypeError("batch_size must be an integer")
```
